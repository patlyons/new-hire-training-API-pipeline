# new-hire-training-API-pipeline

# Python Pandas Data Analysis Portfolio

This repository contains Python API calls designed using the Python tool in Alteryx. These API calls pull data from cloud-based learning management system and create tables that are cleaned and joined in Alteryx to pipeline data to AWS Redshift. The data is then connected to Tableau creating a dahboard showing the lesson completion rate of new hires. This dashboard helps reduce time managers spend each week on empoyee performance evalution. It also helped to decrease the time it takes for new hires to become profienct in their roles.

## Skills Demonstrated

- API Calls
- While loops
- Flatten contents
- User defined functions
- For loops
- Offsets
- Pagination

## API Calls

| Table | Purpose |
|----------|----------|
| Assignment Contents | Flattens json data, creating rows for indivisual assignments that were assinged in batches|
| Assingments| Shows completion rate of batch assignments|
| Users | Shows users and managers|
